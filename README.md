# 老（捞）人活动中心 - 群成员之爹

这个 bot 由羊老师专门为我们频道孤寡老人制作，除了 discord 指南提供的石头剪刀布功能外更提供温馨聊天功能，语音与文字兼备，深受群成员喜爱
